# Smart Expense Splitter 💰

## Overview

Smart Expense Splitter is a simple web application built using Python Flask and basic web technologies. It helps users easily split expenses among friends, groups, or roommates in a quick and organized way.

## Features

* Add multiple expenses
* Split bills among users
* Simple and clean UI
* Fast calculations
* Easy to use interface

## Technologies Used

* Python (Flask)
* HTML
* CSS
* JavaScript

## Project Structure

* app.py → Main backend Flask file
* templates/ → HTML files
* static/ → CSS and JavaScript files
* screenshots/ → Output images of project

## How to Run

1. Install Flask

   ```
   pip install flask
   ```

2. Run the project

   ```
   python app.py
   ```

3. Open in browser

   ```
   http://127.0.0.1:5000
   ```

## Output

Screenshots of the working project are added in the screenshots folder.

## Purpose

This project was created to understand web development basics, backend logic using Flask, and frontend integration.
